Design a modern, professional SaaS-style web UI for a project called "CraftAI Hub".

STYLE:
- Clean, minimal, startup-style (inspired by Stripe, Notion, Vercel)
- Use soft colors, lots of white space
- Rounded corners (16px–24px)
- Soft shadows and smooth hover effects
- Professional typography (Inter or Poppins)

--------------------------------------------------

🏠 HOME PAGE (DASHBOARD)

Create a landing dashboard with:

- Large centered heading: "CraftAI Hub"
- Tagline: "Preserving culture, empowering artisans, and connecting the world through AI"

- A responsive grid of 5 feature cards (2x2 + 1 centered below)

FEATURE CARDS:

1. CraftVision AI
   Description: Identify crafts from images

2. CraftGen AI
   Description: Generate new craft designs

3. CraftMarket AI
   Description: Buy and sell handicrafts

4. CraftAuth AI
   Description: Verify craft authenticity

5. CraftVerse Museum
   Description: Explore cultural heritage

Each card should include:
- Icon (modern line icons)
- Title
- Short description
- Hover effect (scale + shadow)
- Clickable feel (button or card interaction)

--------------------------------------------------

🧭 NAVBAR

Top navigation bar:
- Left: Logo "CraftAI Hub"
- Right: Home | Marketplace | Museum
- Clean, sticky navbar

--------------------------------------------------

📄 PAGE DESIGNS

Create separate UI screens for each feature:

--------------------------------------------------

1️⃣ CraftVision AI (Recognition Page)

- Image upload box (drag & drop style)
- Upload button
- Result card UI showing:
  - Craft Name
  - Origin
  - Description
  - Authenticity Score (progress bar or badge)

--------------------------------------------------

2️⃣ CraftGen AI (Generator Page)

- Dropdown: Select craft type
- Dropdown: Select design style
- Generate button (primary CTA)
- Output section:
  - Generated image preview
  - Download button

--------------------------------------------------

3️⃣ CraftMarket AI (Marketplace Page)

- Grid layout (3–4 columns)
- Product cards:
  - Image
  - Product name
  - Price
  - Buy button

- Optional: filter or category tabs

--------------------------------------------------

4️⃣ CraftAuth AI (Authenticity Page)

- Image upload area
- Verify button
- Result card:
  - Authenticity Score (progress bar)
  - Status badge (Genuine / Fake)

--------------------------------------------------

5️⃣ CraftVerse Museum (Museum Page)

- Grid gallery layout
- Craft cards:
  - Image
  - Name
  - Short description

- On click:
  - Open modal with:
    - Large image
    - Cultural story
    - Origin

--------------------------------------------------
